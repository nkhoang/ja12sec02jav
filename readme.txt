# INSTALLING SASS

(Reference link: http://docs.sencha.com/ext-js/4-0/#!/guide/theming)

1. Download Ruby at: http://rubyinstaller.org/
2. gem install compass
3. Check installation:
  compass -v
  sass -v

  Note: sass may have compatible problem so in order to fix it use these commands:

gem uninstall sass
gem install sass -v 3.1.1
